# Engenharia Informática — Projeto Web

Este é meu **primeiro projeto web**, desenvolvido apenas com **HTML e CSS**, com foco em estruturação de páginas, navegação entre arquivos e estilização visual.

---

## Estrutura do Projeto

```
PROJETO
 ├── index.html
 ├── maisinf.html
 ├── sobre.html
 ├── style.css
 ├── Favicon.png
 ├── Imagem.png
 └── imagem2.png
```

---

## Sobre o Projeto

O site possui um design estilizado e foi criado para praticar:

* Estrutura semântica em HTML
* Estilização com CSS
* Navegação entre páginas
* Organização de arquivos
* Uso de imagens locais

Mesmo sem usar linguagens de programação ou JavaScript, o projeto demonstra como construir um site visualmente organizado e funcional apenas com tecnologias base da web.

---

## Navegação

O site possui **3 páginas HTML interligadas**, permitindo que o usuário navegue entre elas através de links internos.

---

## Como Visualizar

1. Baixe ou clone o repositório
2. Abra o arquivo:

```
index.html
```

3. O site abrirá no navegador padrão.

---

## Objetivo

Este projeto foi desenvolvido como **primeiro passo no desenvolvimento web**, com o objetivo de entender na prática:

* Como páginas web funcionam
* Como arquivos se conectam
* Como criar layouts visuais com CSS
